# Phase 10: Custom Development (Expert Level)

## Topics

- Creating custom nodes
- Extending n8n internals safely
- Writing TypeScript nodes
- Packaging and deployment
- CLI usage

## Hands-on

Build a custom integration node for one internal API.

## Suggested workflow

1. Define node operations and credentials
2. Implement TypeScript node logic
3. Package and publish privately
4. Load extension in n8n and validate

## Success Criteria

- Custom node follows internal API contract
- Credentials and errors are handled safely
- Versioning and release notes are documented
